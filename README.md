# capstone-project
Logistics Management System
Overview
The Logistics Management System is a Java-based application that models a warehouse transportation network using graph data structures. It allows users to manage routes between warehouses, calculate the shortest transportation path, explore all possible routes, and monitor warehouse connectivity. The project demonstrates the practical application of Graph Theory, Dijkstra's Algorithm, and Depth-First Search (DFS) in logistics and supply chain management.
Features
Add new routes between warehouses with transportation costs.
Update existing route costs dynamically.
Delete routes from the network.
Check whether a warehouse exists in the network.
Display the complete warehouse connectivity graph.
Find the shortest path between two warehouses using Dijkstra's Algorithm.
Display all possible paths between warehouses using DFS.
Interactive menu-driven console interface.
